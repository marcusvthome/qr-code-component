## Meu processo

### Tecnologias utilizadas

- HTML5 semântico
- CSS puro
- Flexbox
- Google Fonts

### O que aprendi

Este foi meu primeiro projeto utilizando CSS puro, sem depender de frameworks como Bootstrap.
Aprendi a centralizar elementos com Flexbox, trabalhar com border-radius, cores em HSL
e importar fontes do Google Fonts diretamente no CSS.

### Próximos passos

Quero continuar evoluindo na estilização com CSS puro e dar os primeiros passos
com JavaScript para adicionar interatividade aos projetos.
